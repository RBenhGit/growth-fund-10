# Run Streamlit Application

Launch the main Streamlit financial analysis application.

**Usage**: `/rbh/run-app`

## Steps:

1. Check if Streamlit is installed
2. Locate main Streamlit app file
3. Launch Streamlit server
4. Display local URL for access

## Implementation:

Check for common Streamlit entry points:
- `app.py`
- `streamlit_app.py`
- `main_app.py`
- Or use batch file: `run_fcf_streamlit.bat`

```bash
streamlit run app.py
```

The app will typically open at: `http://localhost:8501`
