# Run FCF Streamlit App

Launch the Free Cash Flow analysis Streamlit application.

**Usage**: `/rbh/run-fcf`

## Steps:

1. Execute the FCF Streamlit batch file or Python script
2. Wait for server to start
3. Display access URL

## Implementation:

On Windows:
```bash
run_fcf_streamlit.bat
```

On Unix/Linux:
```bash
streamlit run fcf_app.py
```

## What It Does:

- Launches the Free Cash Flow analysis dashboard
- Provides interactive financial modeling interface
- Allows DCF and valuation analysis
- Opens browser automatically at http://localhost:8501
