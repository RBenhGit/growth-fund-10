**קריטריונים מחייבים:**
1. **רווחיות יציבה:** רווח נקי חיובי ברצף של 5 השנים האחרונות
2. **יציבות תפעולית:** רווח תפעולי חיובי במינימום 4 מתוך 5 השנים האחרונות

### שלב 3: חישוב וציון מניות הבסיס
חשב ציון לכל אחת מהמניות הנבחרות ודרג אותן בטבלה. הצג בטבלה את **כל** המניות שעברו את סינון שלב 2.

**שיטת חישוב הציון** (ראה פירוט במסמך האפיון):
- **תקופת מדידה:** 3 השנים האחרונות
- **משקולות דירוג:**
  - צמיחה ברווח נקי: 40%
  - צמיחה בהכנסות: 35%
  - גודל החברה (שווי שוק): 25%

**פורמולה:**
```
ציון_סופי = (ציון_צמיחת_רווח × 0.40) + (ציון_צמיחת_הכנסות × 0.35) + (ציון_גודל × 0.25)
```

עבור חישוב מפורט של צמיחה שנתית ונורמליזציה - עיין במסמך האפיון הרלוונטי (סעיף "ציון הכלה בקרן").

#### הסבר תהליך הנורמליזציה

**מדוע צריך נורמליזציה?**
ציוני הצמיחה, ההכנסות ושווי השוק נמדדים ביחידות שונות (%, $) ובסדרי גודל שונים. כדי להשוות ביניהם בצורה הוגנת, יש לנרמל אותם לסולם אחיד של 0-100.

**נוסחת הנורמליזציה:**
```
ציון_מנורמל = ((ערך - ערך_מינימלי) / (ערך_מקסימלי - ערך_מינימלי)) × 100
```

**דוגמה:**
נניח שיש 5 מניות עם צמיחת רווח נקי:
- מניה A: 22.3% (הגבוה ביותר)
- מניה B: 15.7%
- מניה C: 12.5%
- מניה D: 8.2%
- מניה E: -2.1% (הנמוך ביותר)

נורמליזציה:
- מינימום = -2.1
- מקסימום = 22.3
- טווח = 22.3 - (-2.1) = 24.4

חישוב לכל מניה:
- מניה A: ((22.3 - (-2.1)) / 24.4) × 100 = **100.0**
- מניה B: ((15.7 - (-2.1)) / 24.4) × 100 = **72.9**
- מניה C: ((12.5 - (-2.1)) / 24.4) × 100 = **59.8**
- מניה D: ((8.2 - (-2.1)) / 24.4) × 100 = **42.2**
- מניה E: ((-2.1 - (-2.1)) / 24.4) × 100 = **0.0**

**מקרה מיוחד:**
אם כל המניות בעלות אותו ערך בדיוק (מקסימום = מינימום), כולן מקבלות ציון **50.0**.

**חישוב ציון סופי:**
לאחר נורמליזציה, מכפילים כל ציון במשקלו:
```
ציון_סופי = (ציון_רווח_מנורמל × 0.40) + (ציון_הכנסות_מנורמל × 0.35) + (ציון_שווי_מנורמל × 0.25)
```

דוגמה עם מניה C:
- ציון רווח מנורמל: 59.8
- ציון הכנסות מנורמל: 45.2
- ציון שווי מנורמל: 95.1

ציון סופי = (59.8 × 0.40) + (45.2 × 0.35) + (95.1 × 0.25) = 23.92 + 15.82 + 23.78 = **63.52**

### שלב 10: חישוב עלות מינימלית ליחידת קרן

#### עקרון החישוב
המטרה היא למצוא את **העלות המינימלית של יחידת קרן אחת** כך שכל מניה בקרן תופיע ב**מספר שלם** של מניות (לא שברים).

#### שיטת החישוב

**צעד 1: זיהוי המניה היקרה ביותר**
מצא את המניה בעלת המחיר הגבוה ביותר בתיק.

**צעד 2: חישוב עלות בסיסית**
```
עלות_יחידת_קרן = מחיר_המניה_היקרה / משקל_המניה_היקרה
```

**צעד 3: חישוב מספר מניות לכל אחזקה**
```
מספר_מניות = (משקל_המניה × עלות_יחידת_קרן) / מחיר_המניה
```

**צעד 4: עיגול למספרים שלמים**
אם מספר המניות לא יצא שלם, יש לעגל כלפי מעלה באמצעות math.ceil().
**שיטת העיגול (מומלצת):** עיגול כלפי מעלה מבטיח שלא נחסר במספר המניות הנדרש ושומר על עלות קרן סבירה עם סטיות קטנות מהמשקלים היעד.
